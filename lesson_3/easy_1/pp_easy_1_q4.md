```ruby
numbers = [1, 2, 3, 4, 5]
numbers.delete_at(1)
```
The above code will delete the item at index 1 ('2' in this case) from the array and return the item.

```ruby
numbers = [1, 2, 3, 4, 5]
numbers.delete(1)
```
The above code will delete all instances of the object ('1') from the array and return the item.
